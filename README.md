# Collaborative Project Management Tool

## Project Setup

### Prerequisites
- Node.js >= 14.x
- npm or yarn
- Git

### Installation

1. **Clone the repository:**

   ```bash
   git clone <repository-url>
   cd collaborative-project-management
   ```

2. **Install dependencies for both server and client:**

   ```bash
   # Backend
   cd backend
   npm install
   npm run dev
   
   # Frontend
   cd ../frontend
   npm install
   npm run dev
